This property is used to decide whether the element will display in webpage or not. Visibility: Visible | hidden | collapse Visibility: visible -------------> this is the default one Visibility: hidden ------------> It will hide the element and space will be occupy as it is. Visibility: collapse------------> same as hidden but it other element will occupy its space. It is basically used with table.
